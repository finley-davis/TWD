Texas Well Depth Project

Geosciences Visualization of Aquifer Well Depths

Project Overview

This project visualizes groundwater well depths across major Texas aquifers over time using 3D geospatial animations. Each animation represents a different aquifer system, helping to highlight spatial and temporal patterns in groundwater depth.
